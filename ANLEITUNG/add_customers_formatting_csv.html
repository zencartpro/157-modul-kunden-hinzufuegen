Formatieren einer CSV-Datei für Massen-Uploads (Kunden per Admin hinzufügen in Zen Cart 1.5.7 deutsch)

Die Überschriften können in beliebiger Spaltenreihenfolge stehen, solange sie in der ersten nicht leeren Zeile der hochgeladenen Datei enthalten sind.
Die Überschriftenwerte und ihre erwarteten Daten sind:

Titel Beschreibung
gender Wenn Konfiguration->Kundendetails->Anrede auf true gesetzt ist, muss der Wert einer von m oder M (für männlich), f oder F (für weiblich) oder d oder D für Divers sein; ansonsten wird der Wert ignoriert.
first_name Die Mindestlänge wird durch Konfiguration->Minimale Werte->Vorname gesteuert; wenn dieser Wert 0 ist, ist dieses Feld optional.
last_name Die Mindestlänge wird durch Konfiguration->Minimale Werte->Nachname gesteuert; wenn dieser Wert 0 ist, ist dieses Feld optional.
dob Wenn Kundendetails->Geburtsdatum auf true gesetzt ist, wird die Mindestlänge durch Konfiguration->Minimale Werte->Geburtsdatum gesteuert und das Datum muss in dem Format sein, das im Dropdown-Menü "DOB Format" ausgewählt wurde.
email Plichtfeld. Die Mindestlänge wird über Konfiguration->Minimale Werte->E-Mail-Adresse gesteuert. Unabhängig von der Mindestlänge muss der Wert eine "richtige" E-Mail-Adresse sein und darf nicht bereits von einem anderen Kunden in Ihrer Datenbank verwendet werden.
company Wenn Konfiguration->Kundendetails->Firma auf true gesetzt ist, wird die Mindestlänge durch Konfiguration->Minimale Werte->Firma gesteuert; andernfalls wird der Wert ignoriert.
street_address Die Mindestlänge wird durch Konfiguration->Minimale Werte->Strasse gesteuert; wenn dieser Wert 0 ist, ist dieses Feld optional.
suburb Wenn Konfiguration->Kundendetails->Adresszeile 2 auf false gesetzt ist, wird dieses Feld nicht angezeigt; andernfalls wird das Feld angezeigt, ist aber optional.
state Wenn Konfiguration->Kundendaten->Bubdesland auf false gesetzt ist, wird dieses Feld ignoriert; andernfalls muss entweder dieses Feld oder das Feld zone_id angegeben werden. Die Mindestlänge dieses Feldes wird durch Konfiguration->Minimale Werte->Bundesland gesteuert; wenn dieser Wert 0 ist, ist dieses Feld optional. Es kann entweder die Langform oder der Code, z. B. Alabama/AL (Vereinigte Staaten) oder British Columbia/BC (Kanada), verwendet werden.
zone_id Wenn Konfiguration->Kundendetails->Bundesland auf false gesetzt ist, wird dieses Feld ignoriert; andernfalls muss entweder dieses Feld oder das Feld Bundesland angegeben werden. Die numerische Zonennummer, die mit dem Bundesland und Land des Kunden verknüpft ist.
city Die Mindestlänge wird über Konfiguration->Minimale Werte->Stadt gesteuert; ist dieser Wert 0, ist dieses Feld optional.
postcode Die Mindestlänge wird durch Konfiguration->Minimale Werte->Postleitzahl gesteuert; wenn dieser Wert 0 ist, ist dieses Feld optional. Wenn der Wert angegeben wird und das zugehörige Land eines der Länder Kanada, Vereinigtes Königreich oder Vereinigte Staaten ist, muss der Wert dem Postleitzahlenformat für dieses Land entsprechen.
country Dieses Feld ist ein Pflichtfeld und muss einen der zweistelligen ISO-Codes für eines der von Zen Cart unterstützten Länder enthalten.
telephone Die Mindestlänge wird durch Konfiguration->Minimale Werte->Telefonnummer gesteuert; wenn dieser Wert 0 ist, ist dieses Feld optional. Wenn der Wert angegeben wird, muss sein Format dem einer internationalen Telefonnummer entsprechen.
fax Dieses Feld ist optional.
newsletter Gibt an, ob (1) oder nicht (sonst) der Kunde für ein Newsletter-Abonnement registriert werden soll. Wenn das Feld nicht ausgefüllt ist, ist der Kunde nicht für Ihren Newsletter angemeldet.
send_welcome Gibt an, ob (1) oder nicht (0) eine "Willkommens"-E-Mail an den Benutzer gesendet werden soll. Wird das Feld nicht ausgefüllt, wird keine Willkommens-E-Mail gesendet.
customers_group_pricing Dieser Wert wird als Zahl angegeben, die die group_id der "Gruppenpreise"-Gruppe identifiziert, der der Kunde zugeordnet werden soll. Ein Wert von 0 zeigt an, dass der Kunde zu keiner solchen Gruppe gehört, d.h. zur Preisgruppe -keiner--. Wenn diese Spalte in der .csv-Datei nicht angegeben ist, wird dieser Wert für alle hinzuzufügenden Kunden auf 0 gesetzt.
